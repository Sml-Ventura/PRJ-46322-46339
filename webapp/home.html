Sea2Future

Aplicação para Acesso à Base de Dados

 

  Sea2Future é um projeto lançado pela ENIDH que consiste no desenvolvimento de uma embarcação de controlo remoto capaz de telemetria.
 Em parceria com o ISEL, desenvolveu-se um módulo de análise e processamento de dados dos sensores, para os armazenar e demonstrar nesta aplicação.
